std::out_of_range

<stdexcept>
class out_of_range;
Out-of-range exception
exception
logic_error
out_of_range

This class defines the type of objects thrown as exceptions to report an out-of-range error.

This class is designed so that any program, not just the elements of the standard library, can throw it as an exception. Several members of vector, deque, string and bitset throw this error on an out-of-range error.

It is defined as:
1
2
3
4
 
class out_of_range : public logic_error {
public:
  explicit out_of_range (const string& what_arg);
};


Members

constructor
The constructor takes a standard string object as parameter. This value is stored in the object, and its value is used to generate the C-string returned by its inherited member what.

The class inherits the what member function from exception, along with its copy constructor and assignment operator member functions.

// out_of_range example
#include <iostream>
#include <stdexcept>
#include <vector>
using namespace std;

int main (void) {
  vector<int> myvector(10);
  try {
    myvector.at(20)=100;      // vector::at throws an out-of-range
  }
  catch (out_of_range& oor) {
    cerr << "Out of Range error: " << oor.what() << endl;
  }
  return 0;
}


Possible output:

Out of Range error: vector::_M_range_check
