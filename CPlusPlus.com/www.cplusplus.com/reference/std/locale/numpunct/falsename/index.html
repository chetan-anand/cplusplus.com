std::numpunct::falsename

<locale>
string_type falsename() const;
Return string representing false
Returns the string used to represent the boolean value false.

For the standard specialization of char the function returns "false".
For the standard specialization of wchar_t the function returns L"false".

During its operation, the version of this function in the generic template simply calls the virtual protected member do_falsename, which is the member function in charge of performing the actions described above.

Parameters

none

Return value

The string used to represent the boolean value false.
string_type is a member alias of basic_string<charT>, where charT is the template parameter (i.e., the facet's character type).

// numpunct::falsename example
#include <iostream>
#include <locale>
using namespace std;

int main ()
{
  const numpunct<char>& punct_facet = use_facet<numpunct<char> >(cout.getloc());

  cout << "2+2=5 is ";
  if (2+2==5)
	  cout << punct_facet.truename();
  else
	  cout << punct_facet.falsename();

  cout << endl;
  return 0;
}


Output:

2+2=5 is false
