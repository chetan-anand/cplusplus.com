std::allocator_traits

<memory> 
template <class Alloc> struct allocator_traits;
Allocator traits
This template supplies a uniform interface for allocator types.

The non-specialized version provides an interface that allows to use as allocator just any class that provides at least a public member type value_type and public member functions allocate and deallocate.

But any class type can be used as allocator by specializing this class to give it an interface.

Template parameters

Alloc
The allocator type.
