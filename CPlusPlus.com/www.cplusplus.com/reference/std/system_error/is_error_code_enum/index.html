std::is_error_code_enum

<system_error>
template <class T>
struct is_error_code_enum : public false_type {};
error_code enum flag

  • integral_constant
  • is_error_code_enum

This is a traits class to identify whether a particular type is an error code enum type, and thus can be used to construct or assign values to objects of type error_code.

The standard header only provides the default definition, which simply inherits from false_type. But it should be specialized as inheriting from true_type to enable the construction of error_code object from error code enum types.
